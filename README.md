# 📊 Telco Customer Churn - Müşteri Kaybı Tahmini ve API Servisi

## 🎯 Proje Amacı
Bu proje, Telco Customer Churn veri seti üzerinde keşifsel veri analizi (EDA) yapmayı, bir makine öğrenmesi modeli geliştirmeyi ve bu modeli HTTP üzerinden çalışan basit bir API servisi haline getirmeyi amaçlamaktadır.

## 📂 Proje İskeleti (Klasör Yapısı)
Dosya sisteminin karışmaması ve sistemli bir geliştirme süreci için proje aşağıdaki gibi yapılandırılmıştır:
* **`/data`**: Ham ve temizlenmiş veri setlerinin tutulduğu alan.
* **`/notebooks`**: Keşifsel veri analizi (EDA) ve model denemelerinin yapıldığı Jupyter defterleri.
* **`/api`**: Modelin dışarıya açılmasını sağlayan web servisi kodları (FastAPI/Flask).

## 🚀 İş Akışı ve Görev Dağılımı
* **Aşama 1:** Veri Keşfi ve Ön İşleme
* **Aşama 2:** Makine Öğrenmesi Modelinin Kurulması
* **Aşama 3:** Modelin API Haline Getirilmesi

*(Not: Proje geliştirildikçe bu dokümantasyon güncellenecektir.)*
